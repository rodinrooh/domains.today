# Internet Airport

The frontend (what users see) is no longer the `app/` folder in this repo. It moved to `rodinrooh/rodinrooh` and is live at `rodinrooh.com/internet-airport`.

**To update the UI:** edit `app/internet-airport/page.tsx` in the `rodinrooh/rodinrooh` repo, then push — Vercel deploys automatically.

The scraper (`scripts/`), GitHub Actions, and Supabase schema all still live here and still work as-is.
